
# 一、引言

本项目旨在为你提供一个实践平台，将过去几节课学到的新知识融会贯通。本次任务是制作一份虚构服务的注册表单。

## 二、任务要求

### 步骤1：环境搭建与规划

1. 搭建你的Git代码仓库（如果需要回顾操作方法，可参考往期项目）。
2. 创建HTML和CSS文件，并填充一些测试内容，确保所有文件的链接关系正确无误。
3. 下载[设计稿](https://cdn.statically.io/gh/TheOdinProject/curriculum/afdbabfab03fbc34783c6b6f3920aba4a4d3b935/intermediate_html_css/forms/project_sign_up_form/imgs/sign-up-form.png)的高清版本，大致规划好HTML文档中各部分内容的布局方式。

![TOP注册表单项目设计文件图](https://pic.souketsu.com/ObsidianImg/2025/12/TOP注册表单项目设计文件图.png)

## 步骤2：素材准备

1. 设计稿中包含一张大幅背景图，请自行寻找并下载一张你想用的图片作为该区域的背景。设计稿中使用的图片可在[Unsplash网站](https://unsplash.com/photos/25xggax4bSA)找到，你也可以选择其他图片，但务必注明图片创作者的出处！

![[resources/中国风建筑.jpg]]

1. 为“标识”区域挑选一款外部字体。我们使用的是[北欧粗体字](https://cdn.statically.io/gh/TheOdinProject/theodinproject/efdc2888072f409e687d31dc580595dbe4fe0ff4/app/assets/fonts/Norse-Bold.otf)，你也可以任选喜欢的字体。
