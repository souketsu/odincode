# 项目的详细规划和逻辑思路

这是一个非常经典的 The Odin Project 实战项目，它的核心目标是让你从“手写 HTML 结构”转向“用 JavaScript 构建 UI”。这个转变是理解现代前端框架（如 React 或 Vue）的基础。

以下是完成这个项目的详细规划和逻辑思路，分为五个阶段：

---

## 第一阶段：工程化环境搭建 (Webpack)

在这个阶段，你的主要目标是让代码能够“跑起来”，并理解模块化打包是怎么回事。

1. 初始化项目：
    - 在文件夹中运行 npm init 生成 package.json。
    - 安装 webpack 和 webpack-cli。
2. 建立文件结构：
    - 创建 src 文件夹保存源码，dist 文件夹存放打包后的发布版。
    - 核心文件：src/index.js（入口）和 src/template.html（HTML 模板）。
3. 配置 Webpack：
    - 你需要研究 HtmlWebpackPlugin。它能自动把打包后的 JS 脚本插入到你的 HTML 中。
    - 设置 webpack-dev-server，这样你每次改代码浏览器都会自动刷新。
4. 验证环节：
    - 在 index.js 里写一个 console.log('Hello Webpack')。
    - 打包并在浏览器预览，确信你能在控制台看到这句话，环境就算通了。

---

## 第二阶段：核心骨架与第一个页面 (Home Tab)

这个阶段，你将不再在 index.html 里写标签，而是转行“动态编程”。

1. HTML 准备：
    - 在你的 HTML 模板中，只需要保留一个带 ID 的空容器，例如 `<div id="content"></div>`。这就是你所有内容的“底座”。
2. 创建内容生成器：
    - 新建一个文件 src/home.js。
    - 思路：定义一个函数，在函数内部使用 document.createElement() 创建标题、图片、简介文字等。
    - 最后，这个函数应该返回一个包含所有这些元素的“大容器”元素（比如一个 div）。
    - 使用 export default 将这个函数导出。
3. 主逻辑渲染：
    - 在 src/index.js 中 import 你的 Home 生成函数。
    - 获取 HTML 中的 #content 元素。
    - 将生成的 Home 页面元素 appendChild 到 #content 中。

---

## 第三阶段：模块化内容扩展 (Menu & Contact Tabs)

这一阶段是练习“代码复用”和“模块化”。

模仿 Home 结构：
    - 创建 src/menu.js 和 src/contact.js。
    - 遵循同样的逻辑：创建一个函数 -> 使用 JS 创建对应的 DOM 结构 -> 导出该函数。
提示：菜单页可以尝试用循环来创建多个菜品项，提高复用性。

---

## 第四阶段：标签页切换逻辑 (The "Brain")

这是项目的难点，也是最有趣的部分：实现单页应用 (SPA) 的导航逻辑。

导航按钮响应：
        - 在 index.js 中，为导航栏（Navbar）上的三个按钮分别绑定点击事件监听器（Event Listener）。
“清空与重建”模式：
    - 关键思路：当用户点击“菜单”按钮时，你需要执行两步：
        - 清空内容：获取 #content 元素并清除它的所有子元素（例如：content.innerHTML = ''）。
        - 注入新内容：调用 menu.js 里的函数，并将返回的新元素挂载到 #content 上。
优化用户体验：
    - 给当前选中的按钮添加一个特殊的 CSS 类（如 .active），通过 JS 动态切换这个类。

---

## 第五阶段：样式美化 (CSS)

虽然项目要求用 JS 创建元素，但样式依然建议写在 CSS 文件里。

引入样式：
    - 在 Webpack 中配置 css-loader 和 style-loader，然后在 index.js 中 import './style.css'。
设计感提升：
    - 使用 Flexbox 或 Grid 进行页面布局。
    - 为按钮添加悬停效果（Hover Effects）。
    - 考虑使用 Google Fonts 提升整体质感。

---

## 💡 核心知识点提示（不给代码，但给你方向）

- DOM 操作常用方法：createElement, appendChild, classList.add, textContent。
- 图片处理：在 Webpack 中加载本地图片，你需要 import imagePath from './image.png' 然后把 imagePath 赋值给 img.src。
- 模块化语法：理解 import（拿进来）和 export（送出去）的配合。
