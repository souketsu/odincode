# R_P_S 设计文档

## 1.项目介绍

### 1.1 项目名称

中文名称：《剪刀石头布》游戏
英文名称：《Rock Paper Scissors》 Game

### 1.2 项目描述

这是一个简单的石头剪刀布游戏，玩家可以在浏览器中进行游戏。

## 2.游戏规则

1. 玩家需要在浏览器中打开游戏页面。
2. 玩家需要输入自己的姓名。
3. 玩家需要选择石头、剪刀或布。
4. 游戏会随机选择一个玩家和一个电脑。
5. 游戏会根据玩家和电脑的选择，判断胜负。
6. 游戏会显示玩家和电脑的选择，以及胜负结果。
7. 玩家可以选择是否继续游戏。

### 3.0 项目架构图

![项目架构图](RPS_Arch.png)

### 3.1 技术栈

- 前端：HTML、CSS、JavaScript
- 后端：Node.js、Express.js

### 3.2 项目结构

- `RPS_index.html`：游戏页面
- `RPS_ style.css`：游戏页面的样式表
- `RPS_script.js`：游戏页面的 JavaScript 代码
- `RPS_server.js`：游戏服务器的代码

## 4. 项目实现

### 4.1 前端实现

- 游戏页面使用 HTML、CSS、JavaScript 实现。
- 游戏页面包括玩家输入姓名、选择石头剪刀布、显示游戏结果等功能。
- 游戏页面的 JavaScript 代码包括玩家选择石头剪刀布、发送选择到服务器、接收服务器结果、显示游戏结果等功能。
- 游戏页面的样式表包括游戏页面的布局、颜色、字体等样式。
- 游戏页面的布局包括玩家输入姓名、选择石头剪刀布、显示游戏结果等元素的位置。
- 游戏页面的颜色包括背景颜色、字体颜色、按钮颜色等。
- 游戏页面的字体包括游戏标题、游戏选项、游戏结果等字体。
- 游戏页面的字体大小包括游戏标题、游戏选项、游戏结果等字体大小。
- 游戏页面的字体颜色包括游戏标题、游戏选项、游戏结果等字体颜色。
- 游戏页面的按钮包括选择石头剪刀布、继续游戏等按钮。
- 游戏页面的按钮颜色包括选择石头剪刀布、继续游戏等按钮颜色。
- 游戏页面的按钮字体包括选择石头剪刀布、继续游戏等按钮字体。
- 游戏页面的按钮字体大小包括选择石头剪刀布、继续游戏等按钮字体大小。
- 游戏页面的按钮字体颜色包括选择石头剪刀布、继续游戏等按钮字体颜色。

### 4.2 后端实现

- 游戏服务器使用 Node.js、Express.js 实现。
- 游戏服务器包括接收玩家选择、随机选择电脑选择、判断胜负、返回游戏结果等功能。
